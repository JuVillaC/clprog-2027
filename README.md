# 🎸 CL.PROG Festival 2027 - Landing Page

![Estado](https://img.shields.io/badge/Estado-Completado-success)
![Netlify Status](https://img.shields.io/badge/Despliegue-Netlify-00C7B7?logo=netlify)

Una Landing Page inmersiva y de alto rendimiento diseñada para la edición 2027 del festival de metal progresivo **CL.PROG**. Desarrollada con un enfoque "Mobile-First" y estética oscura/industrial.

🔗 **[Ver Sitio en Vivo (Live Demo)](https://clprog-juanvilla.netlify.app/)**

---

## 🛠️ Tecnologías Utilizadas

- **HTML5 & CSS3** (Estructura semántica y diseño Custom Grid/Flexbox).
- **Bootstrap 5.3** (Sistema de grillas, componentes UI e iconos).
- **Vanilla JavaScript** (Lógica de interactividad y temporizadores).
- **Git & GitHub** (Control de versiones).
- **Netlify** (CI/CD Deploy).

---

## 🚀 Características Principales

1. **Diseño Responsivo (Mobile-First):** Adaptable a cualquier dispositivo mediante Media Queries y Bootstrap Grid.
2. **Temporizador Dinámico (JS):** Cuenta regresiva calculada en tiempo real hacia la fecha del festival.
3. **Integración de Terceros (Embeds):**
   - Reproductor oficial de **Spotify** incrustado (Modo Oscuro).
   - Galería dinámica interactiva con posteos reales de **Instagram**.
4. **Navegación Fluida:** Smooth scrolling mediante _href anchors_ para una experiencia UX continua.
5. **Alineación Rítmica (CSS Grid):** Estructura visual coherente para mantener elementos gráficos (como el logo) contenidos armónicamente en todas las resoluciones.
